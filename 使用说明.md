# 使用说明

## 快速开始

### 1. 准备图片

将您的图片文件放入 `public/img/` 文件夹中。

例如:
- `public/img/1.jpg`
- `public/img/2.jpg`

**支持的格式**: JPG, JPEG, PNG, WEBP

### 2. 配置端口

编辑 `public/config/ports.json` 文件:

```json
{
  "ports": [
    {
      "id": "port1",
      "name": "端口 1",
      "qrcodeText": "PORT-001",
      "images": [
        {
          "filename": "1.jpg",
          "watermarkCoordinate": {
            "x": 100,
            "y": 100,
            "width": 150,
            "height": 150
          }
        },
        {
          "filename": "2.jpg",
          "watermarkCoordinate": {
            "x": 200,
            "y": 200,
            "width": 150,
            "height": 150
          }
        }
      ]
    }
  ]
}
```

**配置说明**:
- `filename`: 必须与 `public/img/` 中的文件名完全匹配
- `x`, `y`: 二维码左上角的坐标位置
- `width`, `height`: 二维码的尺寸

### 3. 运行项目

```bash
pnpm dev
```

### 4. 使用工具

1. 打开浏览器访问 http://localhost:5173
2. 选择一个端口
3. (可选) 在文本框中输入自定义二维码内容
4. 点击"生成图片"按钮
5. 等待生成完成
6. 预览生成的图片
7. 点击"下载所有图片 (ZIP)"按钮下载

## 功能说明

### 自定义二维码文本

- 如果输入了自定义文本,将使用该文本生成二维码
- 如果留空,将使用端口配置中的 `qrcodeText` 字段

### 坐标设置技巧

1. **查看图片尺寸**: 使用图片查看器查看图片的宽度和高度
2. **确定位置**: 决定二维码应该放在哪里
3. **设置坐标**: 
   - `x`: 从左边缘开始的像素数
   - `y`: 从上边缘开始的像素数
4. **设置尺寸**: 
   - `width`: 二维码宽度(像素)
   - `height`: 二维码高度(像素)

### 常见问题

**Q: 图片加载失败?**

A: 检查:
1. 图片文件是否在 `public/img/` 文件夹中
2. 文件名是否与配置完全匹配(包括大小写和扩展名)
3. 图片格式是否支持

**Q: 二维码位置不对?**

A: 调整配置文件中的 `x` 和 `y` 坐标值

**Q: 二维码太大或太小?**

A: 调整配置文件中的 `width` 和 `height` 值

**Q: 如何添加更多图片?**

A: 
1. 将图片放入 `public/img/` 文件夹
2. 在配置文件的 `images` 数组中添加新的配置项

      "images": [
        {
          "filename": "1.jpg",

            "y": 50,
            "width": 200,
            "height": 200
          }
        },

          "filename": "2.jpg",
          "watermarkCoordinate": {

            "y": 100,
            "width": 150,
