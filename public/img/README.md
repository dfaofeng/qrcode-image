# 图片文件夹

请将您的图片文件放在此文件夹中。

## 文件命名规则

图片文件名应与 `public/config/ports.json` 配置文件中的 `filename` 字段对应。

### 示例:

如果配置文件中有:
```json
{
  "filename": "port1_image1.jpg"
}
```

则应在此文件夹中放置名为 `port1_image1.jpg` 的图片文件。

## 支持的图片格式

- JPG / JPEG
- PNG
- WEBP

## 注意事项

1. 确保图片文件名与配置文件完全匹配(包括扩展名)
2. 图片尺寸建议不要太大,以免影响处理速度
3. 确保水印坐标不会超出图片边界
